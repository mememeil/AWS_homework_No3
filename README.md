# Name
name-app

## Overview
名前と年齢を登録できるアプリです

## Description
フォームに名前と年齢を入力し「登録」ボタンを押します  
すると、入力した名前と年齢が下部に一覧で表示されます  
登録するたびに、一覧に追加されていきます
登録したデータは、削除、編集が可能です

注意点:
* バリデーションを行っていません
* 名前には文字列、年齢には数値を入力してください
